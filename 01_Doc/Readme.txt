硬件说明：PORTB9=蓝灯，PORTB18=绿灯，PORTB19=红灯

1. 工程概述
(1)本工程为KL25无操作系统应用工程，GPIO样例工程，使得评估板上的一个LED小灯闪烁

2. 软件环境
(1)Kinetis Design Studio IDE v1.1.0.

3. 硬件环境
(1)SD-FSL-KL25-EVB
(2)蓝灯对应引脚为PTB9

4. 实验现象
(1)单步调试情况下，KL25板上的蓝灯闪烁（开始时小灯不闪）
